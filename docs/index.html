What's New in iOS8

Before you begin

This is an iOS8 class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either a configured iOS Simulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This class introduces the new frameworks and features included in iOS8. It is an overview class which touches on the biggest changes in iOS and the developer tools.

Duration

2 hours

Objectives

  1. Explore the new features of iOS8
  2. Identify UIKit API changes
  3. Review and experiment with unified storyboards

Lecture

IOS350 - What's New in iOS8

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Quick look at new frameworks (PhotoKit and Touch ID)
    We will go through two samples showing off the new PhotoKit API which allows you to examine photos which are stored on the device, and the new Touch ID local authentication features which use the touch id sensor. Both samples are included with the lab content.
  2. Working with some of the new UIKit changes in iOS8
    Exercise
    In this first exercise, you will modify an existing application to utilize some of the new features in UIKit such as the UISearchController, new navigation features, and new UITableView capabilities.
  3. Working with Unified Storyboards in Visual Studio
    In this group exercise, we will check out the new unified storyboard support that has been added to XCode and Visual Studio. Together, we will turn on the size class support and show how we can utilize a single storyboard to target different device types and orientations.
